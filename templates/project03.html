Color Detector

The Color Palette Extractor is a Flask-based web application that allows users to upload images and extract a color palette from them. The application uses the ColorThief library to analyze the uploaded image and generate a color palette. Users can visualize the extracted colors and obtain the corresponding hex values.

Key Features:

  • Image Upload:Users can upload images through the web interface.
  • Color Palette Extraction:The ColorThief library is employed to extract a color palette from the uploaded image.
  • Visualization:The extracted color palette is presented to users for visual inspection.
  • Dynamic Interaction:The web application dynamically updates the color palette upon each image upload.

Technologies Used:

  • Flask: The web app is built on the Flask framework, providing a lightweight and modular foundation for the project.
  • ColorThief:A Python library for color extraction from images.
  • Matplotlib:Utilized for plotting and visualizing the color palette.
  • Colormap:Converts RGB values to hexadecimal representation for color visualization.
