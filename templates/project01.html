Online Store

Little Green Heroes is an e-commerce platform that allows users to browse and purchase products from a small family-owned farm. The platform provides a seamless shopping experience, including features for product listing, user registration, cart management, order placement, and secure payment processing.

Key Features:

  • User Registration and Authentication: Users can register for an account with a unique username and email. Passwords are securely hashed using the bcrypt algorithm. Authentication is implemented using Flask-Login, providing a secure and user-friendly login experience.
  • Product Management: Products are listed with details such as name, price, and description. Images associated with each product enhance the visual appeal of the platform.
  • Shopping Cart: Users can add products to their shopping cart, specifying the quantity. Cart contents are stored in the user's session, providing a persistent and personalized shopping experience.
  • Payment Processing: Integration with the Stripe API enables secure and convenient payment processing. Users can make payments using credit/debit cards.
  • User Profile: Users can view and edit their profile information, including contact details and address.

Technologies Used:

  • Flask: The web app is built on the Flask framework, providing a lightweight and modular foundation for the project.
  • Bootstrap: Frontend Styling
  • SQLAlchemy: Database
