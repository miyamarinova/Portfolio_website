Find your daily inspiration for cocktails. Funny website that suggests what cocktail should you drink or prepare today. Can be used many times, for example for a couple of friends who are wondering what to order. The recipes and the images for all the cocktails are collected by using HTTP requests and REST APIs from a public API ('https://www.thecocktaildb.com/api/').
