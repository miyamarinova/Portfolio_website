Random Cocktail

Random Cocktail is a delightful website designed to provide daily inspiration for cocktails. Users can easily discover what cocktail they should drink or prepare today, making it a fun and interactive platform. The project fetches recipes and images for various cocktails by utilizing HTTP requests and REST APIs from a public API ([TheCocktailDB](https://www.thecocktaildb.com/api/)).

Key Features:

  • User-Friendly Interface:The website is designed to be user-friendly, making it easy for individuals or groups to decide on a cocktail choice.
  • Recipe Collection: The platform collects cocktail recipes and images from TheCocktailDB API, ensuring a diverse and extensive selection of beverages.
  • API Integration: Utilizes HTTP requests and interacts with TheCocktailDB API to gather real-time information about cocktails, allowing for an up-to-date and dynamic experience.

Technologies Used:

  • Flask: The web app is built on the Flask framework, providing a lightweight and modular foundation for the project.
  • HTML/CSS:Frontend styling to create an aesthetically pleasing and responsive user interface.
  • REST API (TheCocktailDB):Integration with a public API to dynamically fetch cocktail recipes and images.
  • GitHub Repository:The project is version-controlled and hosted on GitHub for collaboration and version tracking.
