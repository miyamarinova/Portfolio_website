Find your daily inspiration for cocktails. Funny website which suggest you what cocktail should you drink or prepare today. Can be used many times, for example for a couple friends who are wondering what to order. The recipies and the images for all the cocktails are collect by using HTTP request and REST APIs from a public API ('https://www.thecocktaildb.com/api/').
